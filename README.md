# Ultimate Brew Machine

Coffee guide web app built with React + Vite.

## Getting started

Install dependencies:

```sh
npm install
```

Run the dev server:

```sh
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Project structure

- `src/` - React components and app logic
- `public/` - Static assets
